Heightmap erosion map transformations

This list describes transformations of the underlying 2D map object, not every
imperative implementation detail. Parameter setup, loop counters, file I/O, and
other bookkeeping are intentionally omitted.

LSB means least significant bit. In this script, sub-LSB noise means noise
smaller than one 8-bit grayscale level, used to break up banding before and
after the float erosion math.

---

## PRE-EROSION PREPARATION

INPUT: source_png
OUTPUT: prepared_height, conditioned_source_slope

A00 source_png -> source_height
    Function: _erode_heightmap.py:main
    Convert the grayscale source PNG into a float height map in 0..1.
    Snapshot: A00_source_height.png.

A01 source_height -> dequantized_height
    Function: dequantize_height
    Add deterministic sub-LSB dequantization noise to reduce 8-bit banding.
    Snapshot: A01_dequantized_height.png.

A02 dequantized_height -> prepared_height
    Function: prepare_height
    Apply a small Gaussian preblur before slope derivation.
    Snapshot: A02_prepared_height.png.

A03 prepared_height -> raw_source_slope
    Function: source_slope
    Derive slope_x and slope_y because PNG input lacks Buffer A's stored slope channels.
    Snapshots: A03_raw_source_slope_x.png, A03_raw_source_slope_y.png,
    A03_raw_source_slope_magnitude.png before conditioning.

A04 raw_source_slope -> conditioned_source_slope
    Function: limit_slope_magnitude
    Limit slope magnitude to reduce derivative spikes from terraced PNG inputs.
    Snapshots: A04_conditioned_source_slope_x.png, A04_conditioned_source_slope_y.png,
    A04_conditioned_source_slope_magnitude.png after conditioning.

## PORTED SHADER CODE

INPUT: prepared_height, conditioned_source_slope
OUTPUT: eroded_height


S01 prepared_height -> fade_target
    Function: erode_heightmap
    Compute the terrain's peak/valley fade target relative to DEFAULT_HEIGHT.
    Snapshot: S01_fade_target.png.

S02 conditioned_source_slope -> gully_direction
    Function: erode_heightmap
    Convert the conditioned source slope into the initial erosion direction field.
    Snapshots: S02_gully_direction_x.png, S02_gully_direction_y.png,
    S02_gully_direction_magnitude.png.

S03 prepared_height + conditioned_source_slope + fade_target + gully_direction -> octave_gullies
    Function: phacelle_noise
    Evaluate Phacelle-noise gullies across erosion octaves.
    Snapshots: S03_octave_gullies_height.png,
    S03_octave_gullies_slope_x.png, S03_octave_gullies_slope_y.png,
    S03_octave_gullies_slope_magnitude.png.

S04 octave_gullies + fade_target -> faded_gullies
    Function: erode_heightmap
    Mask and fade the raw gully field toward the current terrain context.
    Snapshots: S04_faded_gullies_height.png,
    S04_faded_gullies_slope_x.png, S04_faded_gullies_slope_y.png,
    S04_faded_gullies_slope_magnitude.png.

S05 prepared_height + conditioned_source_slope + faded_gullies -> eroded_height_and_slope
    Function: erode_heightmap
    Accumulate gully height and slope deltas across all erosion octaves.
    Snapshots: S05_eroded_height_and_slope_height.png,
    S05_eroded_height_and_slope_slope_x.png, S05_eroded_height_and_slope_slope_y.png,
    S05_eroded_height_and_slope_slope_magnitude.png.

S06 eroded_height_and_slope - prepared_height -> height_delta
    Function: erode_heightmap
    Extract the accumulated height change.
    Snapshot: S06_height_delta.png.

S07 prepared_height + height_delta -> eroded_height
    Function: erode_heightmap
    Add the accumulated height delta and clamp to 0..1.
    Snapshot: S07_eroded_height.png.

---

INPUT: eroded_height
OUTPUT: export_height_png, rendered_preview

A05 eroded_height -> export_height_png
    Function: dithered_u8
    Dither and quantize the eroded float height map for eroded_<input>.png.

The following steps are performed with a separate script.
They're to aid with visual inspection.

A06 eroded_height -> terrain_colors
    Function: _render_heightmap.py:colorize
    Apply stepped terrain colors for visual inspection.

A07 eroded_height -> hillshade
    Function: _render_heightmap.py:hillshade
    Derive a simple shadow layer from local height differences.

A08 terrain_colors + hillshade -> rendered_preview
    Function: _render_heightmap.py:render
    Combine stepped terrain colors with hillshade for rendered_eroded_<input>.png.
